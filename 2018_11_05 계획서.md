# 2018/11/05
1. 프로젝명 : 스마트 저금통
2. 금주 실습할 내용
	* 실습명 : 데이터 베이스에 있는 값을 읽어오기
	* 실습절차
		* sqlite3을 통해 가상의 통장 생성
		* 통장에 임의의 돈 저축
		* 카카오 플친을 통해 통장 잔고 확인

3. 예상 결과
	* 데이터 베이스의 내용을 자유자제로 사용 사능
	* 카카오톡으로 통장 잔고 , 비밀번호 지정 , 비밀번호 변경 등 다양한 활동 가능
